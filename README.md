## About
cblink/hyperf-socialite 组件衍生于 laravel/socialite 组件的，我们对它进行了一些改造，大部分功能保持了相同。在这里感谢一下 Laravel 开发组，实现了如此强大好用的社会化登陆组件。


## Installing

```shell
$ composer require cblink/hyperf-socialite -vvv
```
